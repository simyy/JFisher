# fisher
a mvc framework for java.

## feature

* mvc
* annotion
* ioc

## sample

This is a sample in src/main/com/simyy/fisher/sample

Run with jetty,
```
mvn jetty:run
```

Visit: http://127.0.0.0.1:4000/

> if you want to use annotion, please set fisrt.

```
// like this
fisher.initDriven("com.simyy.fisher.sample");
```
